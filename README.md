# LoginApp
```
Registers the user using username and password(encrypted)
Generates a Token when user login.
Has ability to change password but is a Protected Action. 
(Meaning the user who is logged in, ie., 
has token available in localstorage can only perform this action,
Can be checked by manually removing token from the localstorage )

```
## Installation

```bash
npm install
node server.js
Visit localhost:5000
```
